PRC training

The repository contains Jupyter Notebooks for the training 'ManGO - Python client' at KU Leuven.  

# How to install

From Linux or WSL:
```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
ipython kernel install --user --name=prc_training
```

# How to run

```
jupyter notebook
```
